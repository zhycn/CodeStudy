# Changelog

格式说明：

- `Added` 新添加的功能
- `Changed` 对现有功能的变更
- `Deprecated` 已经不建议使用，准备很快移除的功能
- `Removed` 已经移除的功能
- `Fixed` 对bug的修复
- `Security` 对安全的改进
